What is Cordova?

A platform for creating mobile apps using HTML+CSS+JS (primarily)

  • Packaged web apps + JavaScript API
  • Provides a bridge to access common native functionalities
  • Uses the default webview for each platform
  • The www folder is (usually) included and served in the app

Why is Cordova awesome?

  • Cordova has a large and active community
  • Plugin architecture means you only get what you use
  • Stable but quick to adapt to changing mobile landscape
  • If needed, it is easy to extend with custom plugins
  • Speed up development of both app logic and UI
  • Web itself has the largest developer community in the world

Cordova vs. Phonegap

  • Phonegap is downstream from Cordova
  • Platform updates and latest plugins are developed for Cordova first
  • Phonegaps primary addition is the cloud build
  • Many plugins don't make it to phonegap build and custom plugins are not supported
  • When using Cordova locally, you have to maintain a build machine and install platform SDK and tools for each native platform

Where to start?

Cordova Docs

Honestly, the docs are well written and very helpful. (make sure you are on the latest version of the docs)

Command Line Interface FTW

Best way to learn and use cordova. If you dont have it, install it right now!


$ sudo npm install -g cordova

Create a HelloWorld app


$ cordova create my_first_app
$ cd my_first_app

Clone these sides


$ git clone git@github.com:KamloopsJavaScript/aras-cdv-demo.git
$ cd aras-cdv-demo/presentation

Try running some of these commands:


$ cordova help
$ cordova platform ls
$ cordova platform add ios
$ cordova serve
$ cordova run
